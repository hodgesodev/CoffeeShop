# CoffeeShop

A tiny coffee shop MVP built with Streamlit and SQLite.

## Sprint 1 Scope

- Cashier can create an order
- Barista can see pending orders
- Customer can see queue position

## Run Locally

```powershell
git clone https://github.com/AbdulA03/CoffeeShop.git
cd CoffeeShop
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
streamlit run app.py
```

## Run Tests

```powershell
pytest -q
```
