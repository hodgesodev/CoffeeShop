## How to set up the environment

1. Clone the repository
```bash
# HTML
$ git clone https://github.com/hodgesodev/CoffeeShop.git

# SSH
$ git clone git@github.com:hodgesodev/CoffeeShop.git

$ cd CoffeeShop
```
2. Create the .env
```bash
# PIP
$ pip -m venv .vemv

# UV
$ uv venv
$ source .venv/bin/activate
```

3. Install required libraries
```bash
# PIP
$ pip install -r requirements.txt

# UV
$ uv pip install -r requirements.txt
```
